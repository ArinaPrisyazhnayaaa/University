# University - Учебный репозиторий

## О репозитории
Этот репозиторий предназначен для организации учебного процесса по направлениям:
- Искусственный интеллект в бизнес-моделях
- Управление ИТ-проектами  
- Управление неструктурированной информацией
- Онтология и моделирование бизнес-процессов
