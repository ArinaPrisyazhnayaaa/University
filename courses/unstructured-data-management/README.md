# Управление неструктурированной информацией

## Описание курса
Изучение методов и технологий работы с неструктурированными данными: тексты, изображения, аудио и видео файлы, их хранение, обработка и анализ.

## Преподаватель
Чалей Иван Вацлавович

## Структура курса
- `materials/` - учебные материалы и лекции
- `assignments/` - задания для студентов
- `resources/` - дополнительные ресурсы

## Темы курса
- Типы неструктурированных данных и их особенности
- Системы хранения неструктурированной информации
- Методы обработки текстовых данных (NLP)
- Компьютерное зрение и обработка изображений
- Анализ аудио и видео контента
- Поисковые системы и индексация
- Data Lakes и Big Data платформы
