# Управление ИТ-проектами

## Описание курса
Изучение методологий, инструментов и практик управления информационно-технологическими проектами на всех этапах жизненного цикла.

## Преподаватель
Чалей Иван Вацлавович

## Структура курса
- `materials/` - учебные материалы и лекции
- `assignments/` - задания для студентов  
- `resources/` - дополнительные ресурсы

## Темы курса
- Основы проектного управления
- Agile и Scrum методологии
- Управление рисками в ИТ-проектах
- Бюджетирование и планирование ресурсов
- Управление качеством и сдачей проектов
- Инструменты управления проектами (Jira, Trello, Asana)
