# Онтология и моделирование бизнес-процессов

## Описание курса
Изучение принципов создания онтологий предметных областей и методов моделирования бизнес-процессов для оптимизации деятельности предприятий.

## Преподаватель
Чалей Иван Вацлавович

## Структура курса
- `materials/` - учебные материалы и лекции
- `assignments/` - задания для студентов
- `resources/` - дополнительные ресурсы

## Темы курса
- Основы онтологического инжиниринга
- Языки моделирования бизнес-процессов (BPMN, UML)
- Принципы построения онтологий предметных областей
- Инструменты моделирования (Bizagi, ARIS, Visual Paradigm)
- Анализ и оптимизация бизнес-процессов
- Реинжиниринг бизнес-процессов
- Внедрение систем управления бизнес-процессами (BPMS)
